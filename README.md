## Installation
To install and set up the project, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/Maidawailtd/mglinkco.git
    ```

2. Navigate to the project directory:
    ```bash
    cd mglinkco
    ```

3. Install the dependencies:
    ```bash
    npm install
    ```
